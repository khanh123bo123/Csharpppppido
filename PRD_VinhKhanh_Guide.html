9. Công nghệ & Lộ trình (Tech Stack)

Bộ công cụ công nghệ và kế hoạch phát triển

Frontend (Mobile)

  • Framework: C# .NET MAUI (Đa nền tảng)
  • Bản đồ: Mapsui / Microsoft.Maui.Controls.Maps
  • Database: Entity Framework Core SQLite
  • Dịch vụ nền: MAUI Essentials (Geofence, TTS)

Backend & Web Portal

  • Framework: ASP.NET Core 8 API / MVC
  • Database: SQL Server
  • Bảo mật: JWT Authentication & Identity
  • Giao diện Admin: Bootstrap 5, Chart.js

Lộ Trình Phát Triển Tương Lai (Phase 3)

  • Map Pack Offline: Tải trước bản đồ (Vector Tiles) để không cần Internet load hình bản đồ.
  • Heatmap Dashboard: Vẽ biểu đồ nhiệt cho Chủ Quán thấy khu vực đông khách theo thời gian thực.
  • AI Guide: Tích hợp OpenAI API để trả lời câu hỏi của khách hàng ngay trên App bằng giọng nói.
